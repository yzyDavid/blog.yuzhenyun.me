参考文档：http://www.pythondoc.com/pythontutorial3/modules.html

以上两种方式会导致sys.path值不一致。作为模块启动时当前目录会被加进 path，否则文件目录加进入 path。造成的影响是：

模块启动 - 可以相对路径导入自己目录的其他模块

文件启动 - 不可以

这指的是导入时搜索的路径不同。

Python 2 - 默认相对路径导入

Python 3 - 默认绝对路径导入
