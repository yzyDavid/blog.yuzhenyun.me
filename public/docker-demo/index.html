Docker Service

yml stack 等概念，要在 swarm 上面部署的服务。Docker 认为服务是同类型容器的组合。一个 stack 当中可以包括多个网络与服务。

注意要在其他物理机上确保镜像存在。哦没主意它会不会自动拉取镜像。
